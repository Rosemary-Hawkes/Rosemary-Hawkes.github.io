How to use classes and IDs in HTML and CSS

Sprint 2 Technical Blog: 22 November 2020

ID's and classes are different

ID’s and Classes allow us to "label" content in an HTML document so that it can be manipulated and styled. They are particularly useful where there is a section on the page that has no relevant tag to signify it, for example a footer, or boxes in a sidebar for keeping content in the sidebar separated.

Classes are not unique but IDs are. The difference between an ID and a class is that an ID can be used to identify one element, whereas a class can be used to identify more than one. You can use the same class on multiple elements and you can use multiple classes on the same element.

An element can have both a class and an ID

An element can have a class or both a class and an ID. An element can also have an ID without a class.

Styling of classes and IDs

You can have the same HTML element but style and present it differently depending on its class or ID. These class and ID labels are referenced on the CSS stylesheet so that they can be styled. In CSS, a class selector is a name preceded by a full stop (“.”) and an ID selector is a name preceded by a hash character (“#”).

Best practise for naming classes and IDs

When naming classes and IDs describe the context of the element, not where it is or what it looks like.

Styling of multiple objects

Any styling information that needs to be applied to multiple objects on a page should be done with a class. The class name will act as the "label" so that the same set of styling can be applied to each of these classes.

Reusable information

Information that is reusable should be kept in a class and information that is totally unique should be kept in an ID. If you need slightly different styling on one of the classes you can still share all the other attributes of that class but apply a second class inside that class attribute.

Unique ID

Because HTML elements can have both a class and an ID you can apply the class for styling all elements of the same class on the page. You would use the unique ID for styling specific to that element.

You can also apply a selector to a specific HTML element by stating the HTML selector first, so p.cat { /* text */ } will only be applied to paragraph elements that have the class “cat”.
